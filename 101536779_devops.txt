COMP 3104 – DevOps for System Administration is a course designed to introduce students to 
the principles and practices of DevOps, focusing on automation, continuous integration/continuous 
deployment (CI/CD), and system administration.

### Key Topics Covered:
- **Version Control (Git & GitHub)** – Managing source code effectively.
- **Continuous Integration & Deployment (CI/CD)** – Using tools like GitHub Actions and Jenkins.
- **Containerization** – Working with Docker and Kubernetes.
- **Infrastructure as Code (IaC)** – Automating infrastructure provisioning with Terraform and Ansible.
- **Monitoring & Logging** – Using tools like Prometheus, Grafana, and ELK stack.

The course emphasizes practical hands-on learning through projects and real-world applications.
